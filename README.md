# IPL Analysis Dashboard

## Overview
This project analyzes IPL match data and provides insights on player performance, match phases, and strategic decisions.

## Features
- Player performance analytics
- Phase-wise analysis (Powerplay, Middle, Death)
- Bowler vs Batsman insights
- Strategic decision engine

## Tech Stack
- Python
- Streamlit
- Pandas, NumPy
- Machine Learning (Random Forest)

## How to Run Locally
```bash
streamlit run app.py
